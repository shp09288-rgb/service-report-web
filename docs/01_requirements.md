# Requirements

## Dashboard
- Create / edit / delete cards
- Search & filter
- Display card status (editable / editing / locked)
- Show current user

## Card
- Site + Equipment 기반
- Installation / Field Service 분리

## Document
- 날짜 기반 보고서
- Internal / External 분리
- History 저장

## Editor
- Section 기반 작성
- Progress (%) 포함
- 이미지 첨부 가능
- 자동 번호 구조

## Installation Only
- Gantt Chart
- Individual Action Chart

## Export
- Word(.docx) 생성
- 다중 문서 병합
- 날짜 필터

## Lock System
- 1명만 편집 가능
- heartbeat 기반 자동 해제

## Backup
- 전체 데이터 export 기능

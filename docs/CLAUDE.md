# Claude Project Instructions

This project is a Park Systems Service Report Web tool.

Before coding:
1. Read all files under `./docs`
2. Read all files under `./references`
3. Use the Excel files in `./references` as the source of truth for report layout and Word export structure
4. Use the zip file in `./references` as the source of truth for UI/UX direction and reusable implementation patterns
5. Do not search outside the workspace
6. Do not use PowerShell to search external folders unless explicitly asked
7. First summarize findings, then propose architecture, then implement step by step
